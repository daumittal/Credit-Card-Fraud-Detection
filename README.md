# Credit-Card-Fraud-Detection
In predictive data analytics, statistical methods are applied to recorded data to forecast future events. R, renowned for its statistical prowess, offers a rich suite of tools for creating predictive models. Its libraries facilitate techniques like regression and time series analysis on diverse datasets, bolstered by robust support for data visualization.

In this project, we'll employ credit card data to detect fraud, undertaking the following steps:

Exploration of the dataset using R's modules for viewing and plotting information.

Development of a logistic regression model for credit card fraud detection.

Construction of a decision tree model with the same objective.

Utilization of the Receiver Operating Characteristic (ROC) curve to compare the performance of the models.

Through this project, we aim to showcase the effectiveness of predictive analytics techniques in identifying fraudulent activities using R's statistical capabilities.
